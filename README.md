Run Lighthouse audit and get AI-powered fix suggestions.

## Install

```bash
npm install -g ai-lighthouse
```

## Usage

```bash
npx ai-lighthouse https://mysite.com
npx ai-lighthouse https://example.com/page
```

## Setup

```bash
export OPENAI_API_KEY=sk-...
```
